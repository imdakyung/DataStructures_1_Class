GCD알고리즘 설명
: GCD(최대공약수)는 두 정수의 공약수 중 가장 큰 값을 의미하며,
유클리드 알고리즘을 통해 효율적으로 구할 수 있다.

유클리드 알고리즘은 두 수 a, b에 대해
a를 b로 나눈 나머지를 반복적으로 계산하는 방식으로 동작한다.

gcd(a, b) = gcd(b, a % b)

이 과정을 나머지가 0이 될 때까지 반복하면,
마지막으로 남은 값이 최대공약수가 된다.

시간복잡도 분석 
: 유클리드 알고리즘은 반복할 때마다 입력 값이 빠르게 감소하는 특징이 있다.

특히 최악의 경우는 입력 값이 피보나치 수열 형태일 때이며,
이 경우에도 각 단계에서 값이 급격히 줄어들기 때문에
반복 횟수는 입력 크기 n에 대해 log n에 비례한다.

따라서 GCD 알고리즘의 시간복잡도는 다음과 같다.

→ O(log n)

실행 특성
: GCD 연산은 매우 빠르게 수행되며,
피보나치 수열 계산과 비교했을 때 전체 실행 시간에 거의 영향을 주지 않는다.

이는 시간복잡도 O(log n)이 매우 효율적이기 때문이며,
실제 프로그램 실행에서도 GCD 연산은 거의 즉시 결과를 반환하는 것을 확인할 수 있다.

결론
: 유클리드 알고리즘은 나머지 연산을 반복하며 입력 값을 빠르게 감소시키므로
시간복잡도는 O(log n)이다.
