본 과제에서는 홀수 크기 N에 대해 2차원 배열을 생성하고, 1부터 N²까지의 정수를 중복 없이 배치하여 가로, 세로, 대각선의 합이 모두 동일한 마방진을 구현하였다.

본 프로그램은 Siamese Method를 이용하여 마방진을 생성하였다.
첫 번째 숫자 1은 첫 행 가운데 열에 배치한다.
이후 숫자는 “오른쪽 위 방향”으로 이동하여 배치한다.
배열 범위를 벗어날 경우 반대편으로 이동한다.
이미 값이 존재하는 경우, 바로 아래로 이동한다.

마방진 출력 :
2차원 배열을 이용하여 마방진을 표 형태로 출력하였다.
%3d를 사용하여 정렬된 형태로 출력되도록 하였다.

합 계산 & 정검 : 
마방진이 올바르게 생성되었는지 확인하기 위해 다음을 계산하였다
가로 합: N개
세로 합: N개
대각선 합: 2개
총 2N + 2개의 합을 출력하여 모든 값이 동일함을 확인하였다.
