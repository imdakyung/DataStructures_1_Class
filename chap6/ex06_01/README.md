본 프로그램은 사용자로부터 정수 `N`을 입력받아, 크기 `N`의 정수 배열을 **동적 할당**으로 생성한 뒤 다양한 연산을 수행한다.

주요 기능은 다음과 같다:

난수로 배열 초기화/ 배열 정방향 출력/ 배열 역방향 출력/ 배열의 총합, 최대값, 최소값 계산

동적 메모리 할당
:사용자 입력 크기 `N`에 따라 배열을 동적으로 생성
메모리 할당 실패 시 예외 처리 수행

난수 생성
:`srand()`를 통해 매 실행마다 다른 난수 생성
0부터 N까지의 랜덤 값 저장

포인터 기반 배열 접근
: 배열 인덱스(`arr[i]`) 대신 포인터(`*(p)`)로 접근
시작 주소부터 끝 주소까지 이동하며 값 출력



정방향 출력
:배열의 첫 번째 요소부터 마지막 요소까지 순차 출력

역방향 출력
:배열의 마지막 요소부터 첫 번째 요소까지 역순 출력

총합, 최대값, 최소값 계산
:하나의 반복문에서 모든 연산 수행
효율적인 탐색 구조

출력 예시
정방향 출력: 3 1 5 2 4
역방향 출력: 4 2 5 1 3
총합: 15
최대값: 5
최소값: 1
