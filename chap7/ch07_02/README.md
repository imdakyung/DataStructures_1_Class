본 프로그램은 구조체 배열을 동적 할당하여 학생 정보를 저장하고,
포인터 연산을 이용하여 성적을 분석하는 프로그램이다.

구현 내용

- 이름 (문자열)
- 학번 (1 ~ N 자동 증가)
- 성적 (1 ~ 100 사이 랜덤 값)

동적 메모리 할당
- 사용자로부터 학생 수 N 입력
- malloc을 이용하여 Student 구조체 배열 생성

이름 입력 (파일 입출력)
- `name_list.txt` 파일에서 학생 이름을 읽어옴
- fgets()를 이용하여 한 줄씩 입력
- 문자열 끝 개행 문자 제거 후 저장

데이터 저장 방식
- 모든 배열 접근은 포인터 연산으로 처리
  - arr[i] 대신 (arr + i)-> 사용

학번 및 성적 생성
- 학번: 1부터 N까지 자동 증가
- 성적: rand() 함수로 1~100 사이 랜덤 값 생성

성적 분석 기능
- 최고 점수 학생 탐색
- 최저 점수 학생 탐색
- 전체 평균 점수 계산

사용 기술
- 동적 메모리 할당 (malloc / free)
- 파일 입출력 (fopen / fgets / fclose)
- 문자열 처리 (strcpy, strcspn)
- 난수 생성 (rand, srand)
- 포인터 연산 (*(arr + i), (arr + i)->field)

실행 방법
1. 프로그램 실행
2. 학생 수 입력
3. name_list.txt 파일에서 이름 자동 로딩
4. 결과 출력 확인
