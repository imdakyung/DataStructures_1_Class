본 프로그램은 동적 메모리 할당을 이용하여 학생 정보를 저장하고,
성적을 분석하는 프로그램이다.

- 이름 (문자열)
- 학번 (1 ~ N 자동 증가)
- 성적 (1 ~ 100 사이 랜덤 값)

2. 동적 배열 생성
- 사용자로부터 학생 수 N 입력
- malloc을 이용하여 Student 구조체 배열 동적 생성

3. 이름 입력
- `name_list.txt` 파일에서 학생 이름을 읽어옴
- fgets를 사용하여 한 줄씩 입력

- 학번: 1부터 N까지 자동 증가
- 성적: rand() 함수로 1~100 사이 난수 생성

- 최고 점수 학생
- 최저 점수 학생
- 전체 평균 점수

- name_list.txt (학생 이름 목록 파일)

- 동적 메모리 할당 (malloc)
- 파일 입출력 (fopen, fgets)
- 문자열 처리 (strcpy, strcspn)
- 난수 생성 (rand, srand)
- 반복문을 통한 최대/최소/합계 계산

1. 프로그램 실행
2. 학생 수 입력
3. name_list.txt 파일에서 이름 자동 로딩
4. 결과 출력 확인
