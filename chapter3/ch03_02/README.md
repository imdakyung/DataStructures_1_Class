두 개 숫자를 입력하면
가장 큰 공약수(GCD)랑
가장 작은 공배수(LCM)를 알려주는 프로그램

GCD는 유클리드 호제법으로 계산하고
LCM은 GCD를 이용해서 계산한다.

GCD(a, b):
b가 0이 될 때까지
temp = b
b = a 나머지 b
a = temp
a를 돌려준다.

LCM(a, b):
a 곱하기 b 나누기 GCD(a, b)



GCD는 두 수가 동시에 나누어지는 가장 큰 수
LCM은 두 수가 동시에 나올 수 있는 가장 작은 수

입력 방법 : 숫자 두 개를 띄어쓰기로 입력

출력 방법 : GCD: 6, LCM: 72
