분수는 숫자를 나눈 형태임
예를 들어 1/2, 3/4

분수는 위에 있는 수를 분자라고 하고, 아래 있는 수를 분모이고
분모는 0이 될 수 없음

그리고 계산을 하면 항상 가장 간단한 형태(기약분수)로 만들어서 출력

자료형 정의 
Fraction
numerator : int
denominator : int (0이 아님)

연산 정의 

1. 사칙연산
분수는 다음 계산을 할 수 있다.

더하기 (+)
빼기 (-)
곱하기 (×)
나누기 (÷)

두 개의 분수를 계산하면 결과도 분수로 나온다.
그리고 항상 기약분수로 바꾼다.


2. 기약분수 만들기
reduce(Fraction f)

분자와 분모를 최대공약수로 나누어서
가장 간단한 분수로 만든다.

예
2/4 → 1/2

또 분모는 항상 양수로 만든다.


3. 문자열을 분수로 바꾸기
from_string(String str)

입력은 여러 가지 형태가 가능하다.

"A/B" → 분수
"A" → 정수 → A/1
"A.B" → 소수

예
"3/4" → 3/4
"5" → 5/1
"0.5" → 1/2


4. 분수 출력하기
print_fraction(Fraction f)

기본은 A/B 형태로 출력한다.
하지만 분모가 1이면 정수로 출력한다.

예
6/1 → 6
3/4 → 3/4



입력 : [분수] [연산자] [분수]

예
2/5 + 2/4



출력 형식 : 기약분수

예
9/10


분모가 0이면 오류가 발생함.
0으로 나누면 오류가 발생.
잘못된 연산자가 들어오면 오류를 출력함.
입력 형식이 잘못되면 오류를 처리함.
