양의 정수 전용 순차 리스트 (ArrayList) 개선

파일 구조
`arrayList.h`: 데이터 구조 정의 및 함수 인터페이스 선언
`arrayList.c`: 순차 리스트의 주요 기능(추가, 삭제, 변경, 확장 등) 구현
`main.c`: 메뉴 기반 사용자 인터페이스 구현

주요 기능
양의 정수 추가 (Insert): 원하는 위치(index)에 양의 정수만 입력 가능하도록 제약 조건 적용.
항목 삭제 (Delete): 특정 위치의 데이터를 삭제하고 리스트를 앞으로 당겨 정렬 유지.
값 변경 (Replace): 기존 위치의 값을 새로운 양의 정수로 변경.
리스트 출력 (Print): 현재 저장된 데이터와 인덱스 정보를 시각적으로 출력.
전체 삭제 (Clear): 리스트의 모든 데이터를 비우고 초기화.

개선 사항
데이터 검증: 입력값이 0 이하인 경우 삽입/변경이 불가능하도록 예외 처리.
동적 할당: 초기 용량이 부족할 경우 `realloc`을 통해 메모리를 자동으로 2배 확장.
안정성 향상: 잘못된 인덱스 접근 시 오류 메시지를 출력하고 프로그램 비정상 종료 방지.
