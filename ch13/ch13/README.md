insertAtLinkedList
특정 위치에 추가: 위치(index)와 정수를 입력받아 해당 위치에 삽입 코드
새로운 노드를 만든 뒤, 포인터를 엉키지 않게 순서대로 바꾸어 리스트 중간에 노드를 끼워 넣는 법
0번째 자리이거나 자리가 리스트에 노드가 없을때
새 노드 만들기: 컴퓨터 메모리에 새로운 노드를 하나 만든다
포인터 연결하기: 새 노드의 포인터가 현재 가장 앞에 있는 헤드 노드를 가리킴
헤드 이름표 바꾸기: 이제 새 노드가 리스트의 맨 앞이 되었으므로, 헤드라는 이름표를 새 노드에게 넘겨줌
크기 늘리기: 리스트의 전체 노드 개수를 1개 늘려줌
중간이나 맨뒤에 넣을때
새 노드 만들기: 새로운 노드를 만든다
원하는 위치 찾아가기: 헤드 노드부터 시작해서 내가 들어가고 싶은 자리의 바로 앞 노드까지 화살표를 따라 차례대로 이동한다
새 노드의 화살표를 앞 노드가 가리키고 있던 다음 노드로 연결하고 앞 노드의 화살표를 새 노드로 향하게 바꾸면 중간에 새노드가 생김
크기 늘리기: 리스트의 전체 노드 개수를 1개 늘려줌


main.c코드 : 이사용자가 번호를 누를 때 컴퓨터 메모리에 노드를 새로 만들고 지우고 순서대로 읽어서 보여주는 것
사용자가 실수로 글자를 입력하면 숫자를 입력해주세요라고 경고한 뒤 입력창을 비움
1번 : 정수를 입력받아 리스트의 가장 앞에 삽입
2번 : 정수를 입력받아 리스트의 가장 뒤에 삽입
3번 : 위치(index)와 정수를 입력받아 해당 위치에 삽입
4번 : 위치(index)를 입력받아 해당 노드를 삭제하고 값 출력
5번 : 현재 리스트에 저장된 모든 정수를 순서대로 출력
6번 : 현재 저장된 노드의 총 개수를 출력
0번 종료: 모든 메모리를 해제(destroy)하고 프로그램을 종료
